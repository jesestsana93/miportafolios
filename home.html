{Why work with me?}

Speed

I make sure that the web loads in the shortest possible time (it depends on the amount of content of the site). If a website takes more than 5 seconds to load, 70% of users close it and do not come back in more.

Databases

If you want dynamic web pages, databases are commonly used, because if the information stored in the database changes, the web page connected to the database will also change accordingly (and automatically) without human intervention.

Mobile priority

More and more people surf from their cell phones anywhere in the world. I make sure that your website is optimized for any type of mobile device.

Self-managing

It is important that you can manage your website yourself, from anywhere and at any time. I develop content management systems that allow you to see the data of your website and edit the content you want, which allows you to stay updated.
